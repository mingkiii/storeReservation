## 개요
매장 테이블 예약 서비스 구현

Use : Spring, Jpa, Mysql

목표 : 사용자가 매장을 방문할 때 미리 예약 서비스로 매장과 소비자에게 편의를 제공하는 서버를 구축한다.

## User api
- [ ] 회원 가입
- [ ] 로그인 토큰 발행
- [ ] 로그인 토큰을 통한 제어 확인 (JWT, Filter를 사용)

## Partner api
- [ ] 회원 가입
- [ ] 로그인 토큰 발행
- [ ] 로그인 토큰을 통한 제어 확인 (JWT, Filter를 사용)
- [ ] 상점 등록

## Store api
- [ ] 상점 검색
- [ ] 상점 상세 정보 확인
- [ ] 상점 예약 목록 보기

## Reservation api
- [ ] user - 예약 요청
- [ ] 예약자 방문 확인(키오스크)
- [ ] partner - 예약 승인, 거절

## Review api
- [ ] review 작성
- [ ] review 조회
